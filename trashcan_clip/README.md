# Mocowanie zastępcze

Orginalny klips małego plastikowego kosza na śmieci się połamal więc został zaprojektowany element mający go zastąpić.

Element miał być lekki i w miarę wytrzynały a przy tym elastyczny aby mógł się zniekształcić w celu nasunięcia na kieszeń drzwi samochodu.

## Wykorzystanie technikaliów druku 3D

Aby osiągnąć założenia skorzystano z możliwości jakie daje Slicer.

Element zaprojektowano tak aby grubość elementu była wielokrotnością grubości zewnętrznych ścian wydruku.

Następnie w slicerze zwiększono ilość zewnętrznych ścian do 4.

W efekcie pocięty model składa się wyłącznie z zewnętrznych ścian bez wypełnienia pomiędzy.

Otrzymany w ten sposób element po całej długości jest ciągłą linią plastiku co daje większą wyrzymałość i elastyczność przy małej grubości.
