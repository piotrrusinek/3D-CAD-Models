# Parametryzowany model elementu lampy

Element miał docelowo być częścią stelaża lampy. Został zaprojektowany jako pojedynczy segment mieszczący się w polu roboczym drukarki 3D z możliwością póżniejszego połączenia kilku elementów w całość.

## Parametryzacja 

Ponieważ projektójąć element nie byłem pewny jakie wymiary będą wizualnie najkorzystniejsze zastowsowoałem zmienne w Fusion 360 aby umożliwić sobie szybkie zmiany w modelu
