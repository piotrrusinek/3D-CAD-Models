# Testowanie funkcji gwintów

Modele zostały stworzone w celu przetestowania funkcji tworzenia gwintów w Fusion 360 i znalezienia odpowiednich parametrów aby gwnit wydrukowany na drukarce 3D był funkcjonalny. 
Mając na uwadze niedokładność druku 3D należało dodać do gwintu odpowiedni offset i zaokrąglenie krawędzi gwintu.
