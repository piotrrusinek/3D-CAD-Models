# Edycja mocowania

Model mocowania telefonu został pobrany z repozytorium modeli a następnie zedytowany.

Model był w formacie .stl więc w oprogramowaniu Fusion 360 został załadowany jako siatka mesh następnie został przekonwertowany do etycji.

Najpierw "naprawiono" ściany. Korzystając z faktu że Fusion 360 automatyczne scala płaszczyzny usuwano kolejne pomniejsze fragmenty płaszczyzn, przez co sąsiadujące fragmenty sie scalały co w końcowym efekcie dało jedną jednolitą płaszczyznę nie podzieloną na trójkąty.

Efekty widać w pliku .f3d gdzie dolna krawędź jest podzielona a pozostałe płaszczyzny nie.

W pliku .stl który jest inną rewizją mocowania dodano otwory na mocowane na gorąco wkładki gwintowane zwymiarowane zgodnie z zaleceniami i wystające piny orientujące.
